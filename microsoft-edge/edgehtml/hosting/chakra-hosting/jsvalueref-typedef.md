# Typedef JsValueRef Typedef

Référence à une valeur JavaScript.  
  
## Syntaxe  
  
```cpp 
typedef JsRef JsValueRef;  
```  
  
## Remarques  
 Une valeur JavaScript est l’un des types suivants de valeurs: undefined, NULL, Boolean, String, Number ou Object.  
  
## Conditions requises  
 **En-tête:** jsrt. h
