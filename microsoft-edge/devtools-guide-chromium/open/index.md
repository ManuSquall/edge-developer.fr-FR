## <span data-ttu-id="39b6b-134">Ouverture automatique DevTools sur chaque nouvel onglet</span><span class="sxs-lookup"><span data-stu-id="39b6b-134">Auto-open DevTools on every new tab</span></span>

### [<span data-ttu-id="39b6b-136">CMD (Windows)</span><span class="sxs-lookup"><span data-stu-id="39b6b-136">CMD (Windows)</span></span>](#tab/cmd-Windows/)

```  

### [<span data-ttu-id="39b6b-137">PowerShell (Windows)</span><span class="sxs-lookup"><span data-stu-id="39b6b-137">PowerShell (Windows)</span></span>](#tab/powershell-Windows/)

```
